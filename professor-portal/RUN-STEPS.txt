تشغيل الموقع محليًا (Professor Portal)

1) افتح PowerShell
2) ادخل إلى مجلد المشروع:
cd "c:\Users\HP BEL INFO\Desktop\test skills\professor-portal"

3) شغّل السيرفر:
npm run dev

4) افتح الرابط الذي يظهر في التيرمنال، غالبًا:
http://127.0.0.1:5173
أو
http://127.0.0.1:5174

ملاحظات مهمة:
- لا تغلق نافذة التيرمنال أثناء العمل، لأن الموقع سيتوقف.
- إذا لم يفتح تلقائيًا، انسخ الرابط من التيرمنال وافتحه يدويًا في المتصفح.
- لإيقاف الموقع: اضغط Ctrl + C داخل نافذة التيرمنال.
